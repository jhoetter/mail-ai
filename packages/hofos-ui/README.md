# @mailai/hofos-ui

Built hofOS UI surface for MailAI. The package is consumed from GitHub Release tarballs, for example:

```json
"@mailai/hofos-ui": "https://github.com/jhoetter/mail-ai/releases/download/v0.1.0/mailai-ui-0.1.0.tgz"
```

Run `pnpm --filter @mailai/hofos-ui build:dist` to produce the library-mode `dist/` artifact.
