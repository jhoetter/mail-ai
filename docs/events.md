# Events

MailAI publishes events to the OS bus.

| Event | When |
|---|---|
| `mailai:email_received` | An inbound email is synced. |
| `mailai:email_sent` | An outbound email is accepted for delivery. |
| `mailai:thread_tagged` | A thread's tags change. |
| `mailai:calendar_event_created` | A calendar event is created. |
